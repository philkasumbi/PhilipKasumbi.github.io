Jumia Flash Sale Scraper

Built an automated scraper using Python, Selenium, BeautifulSoup, and webdriver-manager to extract Jumia Kenya’s flash sale data every 4 hours. Captures product details like prices, discounts, stock levels, and time left, then emails the data as a CSV using SMTP. Scheduled with cron jobs for background execution and solved challenges like pagination, email auth, and driver management.

Kenya Government Tenders Dashboard

Kenya Tenders Dashboard (2025) is an interactive data visualization project built with Python, Pandas, Plotly Express, and Tailwind CSS to analyze and present public tender data from Kenya. It highlights key procurement insights such as tender counts by category and method, publishing-to-closing durations, addendum distributions, and the top procuring entities. The dashboard leverages Jupyter Notebook for data wrangling, embeds dynamic Plotly charts in HTML, and is optimized for web viewing. This project promotes transparency and data-driven understanding of Kenya’s procurement landscape.
